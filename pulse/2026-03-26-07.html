Claude Is Coming for OpenClaw — But OpenClaw Isn't Dead Yet

The Information

Anthropic has been rapidly shipping features that made OpenClaw a sensation: computer takeover for complex tasks, messaging app integration (Telegram, iMessage), memory between sessions, and scheduled tasks. Claude Cowork is easier to install (desktop app vs Terminal commands) and runs in a VM sandbox for better security.

But OpenClaw hit 400,000 daily NPM downloads on Tuesday — near its all-time high. The key advantage: model flexibility. Users can run open-source models like Qwen locally, keeping sensitive files off corporate servers. If open-source models close the performance gap, OpenClaw has "a definite advantage," says Jo's CMO Kevin Li.

"Our software engineers 100% use coding agents now. Many of them haven't generated a line of code in a while, but they're super productive."

For Hiro: This is the competitive landscape for personal AI agents — exactly what Hiro is navigating. The insight: distribution (Claude's ease of install) vs flexibility (OpenClaw's model choice) is the core tradeoff. For financial AI, the local/private model option matters when handling sensitive data.

Jeff Bezos Raising $100B to Buy and Automate Manufacturing

The Information

The two sides of AI's equation are finally aligning: demand for AI capabilities is catching up to supply of compute. The long-term case for AI has strengthened, which justifies the massive data center investments.

Bezos is raising $100 billion — one of the biggest funds ever — to acquire manufacturing companies and automate them with AI. Meanwhile, Anthropic is in talks for a JV with Blackstone and Hellman & Friedman. OpenAI is doing similar deals with TPG, Brookfield, and Bain.

"If there wasn't an AI bubble last year, there might be one this year."

For Hiro: The PE angle is fascinating for Hiro. These firms have captive portfolios that NEED AI transformation to justify valuations. That's a massive enterprise sales channel if you can crack it. The $3T infrastructure spend through 2030 "doesn't sound as bonkers as it did a few months ago."
