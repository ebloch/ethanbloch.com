Privacy Policy

Last updated: February 23, 2026

Overview

Jane is a personal AI assistant application designed for single-user, private use. This privacy policy explains how the application accesses and handles data from Google services.

Data Collection & Use

Jane accesses the following Google services on behalf of its single authorized user:

  • Google Calendar — To read and create calendar events for scheduling and daily briefings
  • Gmail — To read emails for inbox summaries and notifications
  • Google Drive — To access and manage personal documents
  • Google Contacts — To look up contact information

Data Storage

Jane operates locally on the user's personal computer. Data retrieved from Google APIs is:

  • Processed in real-time for the user's requests
  • Not stored on external servers
  • Not shared with third parties
  • Not used for advertising or marketing purposes

OAuth refresh tokens are stored securely in the local system keychain.

Data Sharing

Jane does not share, sell, or transfer any user data to third parties. All data processing occurs locally for the sole benefit of the authorized user.

Data Retention

Data from Google services is accessed on-demand and not permanently stored. Users can revoke access at any time through their Google Account permissions.

Security

Jane uses OAuth 2.0 for secure authentication with Google services. The application:

  • Never stores Google account passwords
  • Uses industry-standard encryption for API communications
  • Stores credentials in the system's secure keychain

User Rights

As the sole user of this application, you have full control over your data:

  • Revoke access at any time via Google Account settings
  • Delete local credentials by removing the application
  • Request information about data access by contacting the developer
